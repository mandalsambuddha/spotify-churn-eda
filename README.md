# Spotify Churn Analysis (EDA)

## 📌 Project Overview
This project performs Exploratory Data Analysis (EDA) on a music streaming dataset to understand user behavior and identify factors contributing to customer churn.

## 🎯 Objective
To analyze user activity and identify key features that influence whether a user continues or stops using the platform.

## 📊 Key Analysis Performed
- Data Cleaning and Preprocessing
- Handling Missing Values and Duplicates
- Feature Engineering (Engagement Score)
- Visualization using different charts:
  - Histogram
  - Scatter Plot
  - Box Plot
  - Count Plot
  - Pie Chart
  - Line Plot
  - Violin Plot
  - KDE Plot
  - Correlation Heatmap
  - Pair Plot

## 🔍 Key Insights
- User engagement (listening time & songs played) strongly affects churn
- Higher skip rates indicate lower user satisfaction
- Subscription type plays an important role in retention
- Device usage shows variation in churn behavior

## 🛠️ Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 📈 Conclusion
The analysis highlights that improving user engagement and personalization can significantly reduce churn. Businesses can use these insights to enhance user experience and increase long-term retention.
